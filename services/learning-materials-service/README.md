# Learning Materials Service

Owns the immutable curriculum:
- Modules, lessons, exercises and their answers/grading criteria.
- Learning paths (immutable once generated - adapting means generating a new path, never editing the old one).
- Assessment module: fixed, deterministic question bank used for initial level assessment.

Does not own: learner-specific/dynamic data (progress, attempts, mistakes - Memory & Progress Service).

See root `README.md` sections 7.2, 8.2.1, 8.2.2.
